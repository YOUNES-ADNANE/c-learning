# 🚀 C Programming Journey - Level 1 (Fundamentals)

Welcome to my first repository in C programming! In this project, I documented my progress step-by-step, from the basics to building a functional logic-based application.

## 📂 Repository Structure & Lessons Learned

Each folder represents a specific milestone in my learning process:

### 🟢 Phase 1: The Basics
*   **ex00 (Hello World):** Setting up the environment and printing my first output.
*   **ex01 (Variables):** Learning how to declare and initialize different data types (int, char).
*   **ex02 (Input):** Using `scanf()` to take data from the user and interact with the program.
*   **ex03 (Arithmetic):** Performing mathematical operations (addition, multiplication, etc.).

### 🔵 Phase 2: Control Flow (Logic)
*   **ex04 (Conditions):** Mastering `if`, `else if`, and `else` to make the program take decisions.
*   **ex05 (Calculator Project):** Combining arithmetic and logic to build a basic calculator.
*   **ex06 (Switch Case):** Learning a cleaner way to handle multiple constant conditions.

### 🟡 Phase 3: Mastering Loops
*   **ex07 (While Loop):** Understanding repetition and the "Infinite Loop" trap.
*   **ex08 (For Loop):** Learning the shorthand way to write loops for cleaner code.
*   **ex09 (Do-While Loop):** Implementing a loop that runs at least once before checking the condition.

### 🔴 Phase 4: Functions & Organization
*   **ex10 (Break & Continue):** Mastering control within loops to skip or exit iterations.
*   **ex11 (Functions):** Learning how to write reusable blocks of code.
*   **ex12 (Void Functions):** Differentiating between functions that return a value and those that just perform a task.
*   **ex13 (Modular Code):** Organizing the project into multiple files using `#include` for custom `.c` files.

### 🏆 Level 1 Graduation Project
*   **ex14 & ex15 (Password Checker App):** 
    This is my final project for Level 1. I combined **Strings (`strcmp`)**, **Infinite Loops (`do-while`)**, and **Conditional Logic** to create a secure login system that keeps asking the user for a password until the correct one is entered.
